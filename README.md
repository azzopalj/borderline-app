# Borderline — App Website

This repo hosts the website for [Borderline](https://apps.apple.com/app/id6772781356), a baseball umpire accuracy game for iOS.

## Pages

- `index.html` — Landing page
- `privacy-policy.html` — Privacy policy (required for App Store)

## Setup

1. Push this repo to GitHub
2. Go to Settings → Pages
3. Set source to "Deploy from a branch" → main → / (root)
4. Your site will be live at `https://yourusername.github.io/borderline-app`

## Privacy Policy URL for App Store Connect

`https://yourusername.github.io/borderline-app/privacy-policy.html`
